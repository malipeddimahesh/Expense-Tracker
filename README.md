# Expense-Tracker

A simple console-based Expense Tracker built using Python. This project helps users record, manage, and analyze their daily expenses while storing data permanently using file handling.

## Features

- Add new expenses
- View all expenses
- Calculate total spending
- Search expenses by category
- Delete an expense
- Display category-wise spending
- Find the highest spending category
- Save expenses to a file
- Load expenses from a file

## Technologies Used

- Python
- Object-Oriented Programming (OOP)
- File Handling

## How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Navigate to the project folder

```bash
cd Expense-Tracker
```

3. Run the program

```bash
python main.py
```
